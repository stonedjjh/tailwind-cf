# Notas del curso

## Algunos Selectores

- **bg** para el background

- **text** para el texto

  - **center** para centrar ejemplo text-center
  - **color-intensidad** para colocar el color de texto ejemplo text-blue-500

- **border** para los bordes

- **hover:** para los estados hover

## Color

### Tranparencia

Para agregar transparencia agregamos / y el numero del porcentaje de transparencia que queremos aplicar

- Ejemplo background color pink 500 con 25% de transparencia:

```css
bg-pink-500/25
```

### Agregar color personalizado

Para agregar un color personalizado podemos usar los corchetes [] tailwind reconoce que todo lo que este dentro de los corchetes es una propiedad personalizada

- Ejemplo agregar un color personalizado de fondo:

```css
bg-[#1877f2]
```
